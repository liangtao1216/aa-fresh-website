AA Fresh Website Setup Instructions
====================================

Your website has been created in the folder: aa-fresh-website

Files included:
- index.html (main website file)
- styles.css (styling and design)
- README.txt (this file)

To complete your website:

1. ADD YOUR LOGO:
   - Save your logo file as "logo.png" in the aa-fresh-website folder
   - If your logo has a different format (jpg, svg), update line 13 in index.html to match

2. UPDATE CONTACT INFORMATION:
   - Open index.html
   - Find the Contact section (around line 110)
   - Add your actual phone number and complete address

3. ADD GOOGLE MAPS (optional):
   - Go to Google Maps (https://www.google.com/maps)
   - Search for your store location
   - Click "Share" > "Embed a map"
   - Copy the iframe code
   - Replace the map-placeholder div in index.html with your iframe code

4. CUSTOMIZE HOURS:
   - Update the business hours in the Contact section if different

5. TO VIEW YOUR WEBSITE:
   - Simply double-click index.html to open it in your web browser
   - Or drag and drop index.html into a browser window

6. TO PUBLISH ONLINE:
   - Upload all files to a web hosting service
   - Or use free hosting like GitHub Pages, Netlify, or Vercel

The website is:
- Fully responsive (works on mobile, tablet, and desktop)
- Modern and professional design
- Easy to customize
- Ready to deploy

Questions? Feel free to ask for help with any customizations!
